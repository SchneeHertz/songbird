# Songbird

<img src="https://raw.githubusercontent.com/SchneeHertz/songbird/master/public/icon.png" alt="icon.png" width="256"/>

一个基于标签的图片管理软件

## 功能
1. 给图片打多个标签
2. 按标签搜索
3. 使用Deepdanbooru自动打标签
4. 内置图片浏览器
5. 关联外置图片浏览器

## 细节
1. 支持多个图库
2. 不影响图库内文件
